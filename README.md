LinePOS
=======

Point Of Service made easy

SetUp
------
### Dependencies
* node.js
* git

### Install and run
    sudo npm install -g bower http-server
    git clone git@github.com:linepos/linepos.git
    cd linepos
    bower install
    http-server app
